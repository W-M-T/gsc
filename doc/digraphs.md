Supported escape sequences

| Sequence | meaning |
| -------- | ------- |
| \0 | null terminator |
| \a | alarm |
| \b | backspace |
| \f | formfeed page break |
| \n | new line |
| \r | carriage return |
| \t | horizontal tab |
| \v | vertical tab |
| \\\\ | backspace |
| \\' | single quote |
| \\" | double quotes |
